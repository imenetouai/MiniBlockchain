 Mini Blockchain en Python

Un projet qui crée une petite blockchain éducative.

 Comment utiliser

1. Télécharge le fichier `blockchain.py`
2. Ouvre un terminal dans le dossier
3. Tape : `python blockchain.py`

Ce que fait le code

- Crée une blockchain simple
- Mine des blocks avec preuve de travail
- Vérifie si la blockchain est valide
- Détecte les modifications
